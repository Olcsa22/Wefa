-Git letöltése (https://git-scm.com/downloads)
-GitHubDesktop letöltése (https://desktop.github.com/)
-Projekt leklónozása (https://github.com/Olcsa22/Wefa.git)
-DbGate letöltése (https://dbgate.org/download/)
-IntelliJ IDEA Ultimate letöltése (https://www.jetbrains.com/idea/download/#section=windows)
